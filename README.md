# RubyRandomDice
サイコロの目をランダムに出力

## 処理
`rand()`メソッドを使用して1 ～ 6のランダムな数字を使って、「サイコロの目は**です」  と出力します。

## コード
```
num = rand(1..6)
puts "サイコロの目は#{num}です"
```

## 出力例  
```
サイコロの目は6です
```
